# Airbnb Price Prediction

## Overview

This project focuses on predicting Airbnb listing prices using machine learning. My contribution includes Exploratory Data Analysis (EDA) and a Linear Regression baseline model.

## Dataset

Kaggle Airbnb dataset:
https://www.kaggle.com/datasets/rudymizrahi/airbnb-listings-in-major-us-cities-deloitte-ml/data

* ~74K listings
* Target: `log_price`

## EDA

* Analyzed price distribution
* Handled missing values
* Explored key features (accommodates, bedrooms, location)
* Visualized relationships and correlations

## Linear Regression Results

* RMSE: 0.416
* MAE: 0.302
* R²: 0.662

## Key Insights

* Property size and location strongly influence price
* Entire homes are more expensive than private/shared rooms
* Geographic features are highly impactful
